My Projects

Educapture

This project is an Android application prototype to improve students learning in classroom. We add functionalities to the open-source project NeutriNote, to evluate a student's note-taking and provide feedback for both the student and the professor.

Image Quantization

Automatization of the determination of an optimal color palette for an image by reducing the number of colors while preserving the original colors' representativeness.

Ongoing research on analysis methods, statoscal methods, and machine learning models to improve table tennis performance. Using a database from the 2024 Olympic Games (53 matches, 17 players). The final objective is to share analysis with the French Table Tennis Federation.
